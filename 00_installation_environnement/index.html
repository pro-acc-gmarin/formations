[WINDOWS] Installation de l'environnement de développement

Versionnement avec Git

Téléchargement

  • Rendez-vous sur le site officiel de Git : https://git-scm.com/
  • Téléchargez la dernière version de Git pour Windows.

Installation

  1. Exécutez le fichier téléchargé.
  2. Suivez les instructions d'installation par défaut.
  3. Une fois l'installation terminée, ouvrez une nouvelle fenêtre d'invite de commande (cmd) et tapez : bash git --version

Compte GitHub

Création

  • Rendez-vous sur : https://github.com/ et créez un compte.

Configuration de Git pour GitHub

  1. Ouvrez l'invite de commande.
  2. Configurez votre nom d'utilisateur et votre adresse e-mail avec les commandes suivantes : bash git config --global user.name "VotreNom" git config --global user.email "votre@email.com"

Installation de Java

Oracle JDK

  • Rendez-vous sur le site officiel d'Oracle : https://www.oracle.com/java/technologies/javase-jdk11-downloads.html
  • Téléchargez la dernière version du JDK pour Windows.

OpenJDK

  • Vous pouvez également utiliser OpenJDK, une alternative open-source à Oracle JDK.
  • Rendez-vous sur le site officiel d'OpenJDK : https://jdk.java.net/
  • Téléchargez la dernière version du JDK pour Windows.

Installation du JDK

Installation du JDK Oracle

  1. Exécutez le fichier téléchargé.
  2. Suivez les instructions d'installation par défaut.

Installation du JDK OpenJDK

  1. Exécutez le fichier téléchargé.
  2. Décompressez le fichier ZIP dans le répertoire de votre choix, par exemple C:\Program Files\Java\.
  3. Configurez les variables d'environnement pour pointer vers le dossier du JDK décompressé :
  4. Cliquez droit sur "Ce PC" > "Propriétés" > "Paramètres système avancés" > "Variables d'environnement".
  5. Sous "Variables système", cliquez sur "Nouveau".
  6. Nom de la variable : JAVA_HOME
  7. Valeur de la variable : Chemin d'installation du JDK.
  8. Sous "Variables système", trouvez la variable Path, puis cliquez sur "Modifier".
  9. Ajoutez %JAVA_HOME%\bin à la liste des valeurs.

Vérification de l'Installation de Java

  1. Ouvrez une nouvelle fenêtre d'invite de commande.
  2. Tapez java -version et javac -version pour vérifier que Java et le compilateur Java ont été installés correctement.

Installation de Apache Tomcat

Téléchargement de Apache Tomcat

  • Rendez-vous sur le site officiel de Apache Tomcat : https://tomcat.apache.org/
  • Téléchargez la dernière version de Tomcat pour Windows (fichier .zip).

Installation de Apache Tomcat

  1. Exécutez le fichier téléchargé.
  2. Décompressez le fichier ZIP dans le répertoire de votre choix.
  3. Renommez le dossier décompressé en apache-tomcat pour faciliter l'accès.

Démarrage de Apache Tomcat

  • Naviguez vers le dossier bin de Tomcat.
  • Double-cliquez sur startup.bat pour démarrer Tomcat.

Configuration des Variables d'Environnement pour Tomcat

  1. Ajoutez le chemin d'installation de Tomcat aux variables d'environnement :
  2. Cliquez droit sur "Ce PC" > "Propriétés" > "Paramètres système avancés" > "Variables d'environnement".
  3. Sous "Variables système", cliquez sur "Nouveau".
  4. Nom de la variable : CATALINA_HOME
  5. Valeur de la variable : Chemin d'installation de Tomcat.
  6. Sous "Variables système", trouvez la variable Path, puis cliquez sur "Modifier".
  7. Ajoutez %CATALINA_HOME%\bin à la liste des valeurs.

Installation de MySQL

Téléchargement de MySQL

  • Rendez-vous sur le site officiel de MySQL : https://dev.mysql.com/downloads/mysql/
  • Téléchargez la version de MySQL Community Server pour Windows.

Installation de MySQL

  1. Exécutez le fichier téléchargé.
  2. Sélectionnez "Custom" (Personnalisé) pour choisir les composants à installer.
  3. Suivez les instructions d'installation par défaut, en sélectionnant les composants nécessaires (MySQL Server, MySQL Workbench, etc.).

Configuration de MySQL

  • Notez le mot de passe root défini lors de l'installation, il vous sera demandé lors de la configuration et de l'utilisation de MySQL.

Vérification de MySQL

  1. Ouvrez une nouvelle fenêtre d'invite de commande.
  2. Tapez mysql -u root -p et entrez le mot de passe root lorsque vous y êtes invité.
  3. Tapez SHOW DATABASES; pour vérifier que MySQL est installé et fonctionne correctement.

Installation de PostgreSQL

Téléchargement de PostgreSQL

  • Rendez-vous sur le site officiel de PostgreSQL : https://www.postgresql.org/download/
  • Téléchargez la version de PostgreSQL pour Windows.

Installation de PostgreSQL

  1. Exécutez le fichier téléchargé.
  2. Suivez les instructions d'installation par défaut.
  3. Lors de l'installation, définissez le mot de passe pour l'utilisateur postgres.

Configuration de PostgreSQL

  • Pour accéder à PostgreSQL via la ligne de commande, ajoutez le répertoire bin de PostgreSQL à votre variable d'environnement Path.

Vérification de PostgreSQL

  1. Ouvrez une nouvelle fenêtre d'invite de commande.
  2. Tapez psql -U postgres et entrez le mot de passe de l'utilisateur postgres lorsque vous y êtes invité.
  3. Tapez \l pour lister les bases de données et vérifier que PostgreSQL est installé et fonctionne correctement.
