Dépendances Maven

Tomcat API

  • Ajout de la dépendance dans le pom.xml:
    <dependency>
        <groupId>org.apache.tomcat</groupId>
        <artifactId>tomcat-servlet-api</artifactId>
        <version>${org.apache.tomcat.version}</version>
    </dependency>

Lombok

Utilisez lombok pour faciliter la génération des différentes méthodes de classes. (Constructeur, Setter/Getter, Hash, ToString, etc..) - Ajout de la dépendance dans le pom.xml:

```
<dependency>
  <groupId>org.projectlombok</groupId>
  <artifactId>lombok</artifactId>
  <version>${org.projectlombok.version}</version>
  <scope>provided</scope>
</dependency>
```

  • Ajout dans la configuration du compiler maven :

    <annotationProcessorPaths> <path> <groupId>org.projectlombok</groupId> <artifactId>lombok</artifactId> <version>${org.projectlombok.version}</version> </path> </annotationProcessorPaths>

MapStruct

Utilisez la librairie MapStruct pour la création des mappers. - Ajout de la dépendance dans le pom.xml:

``` 
<dependency>
  <groupId>org.mapstruct</groupId>
  <artifactId>mapstruct</artifactId>
  <version>${org.mapstruct.version}</version>
</dependency>
```

  • Ajout dans la configuration du compiler maven :

    <annotationProcessorPaths> <path> <groupId>org.mapstruct</groupId> <artifactId>mapstruct-processor</artifactId> <version>${org.mapstruct.version}</version> </path> </annotationProcessorPaths>

IntellIJ

Pour activer la génération des annotations : aller dans Settings > Annotation Processor et activer l'option.
